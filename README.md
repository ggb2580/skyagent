# AI Flight Agent ✈️

> 智能机票查询、预订与订单管理助手

## 📋 项目简介

AI Flight Agent 是一款专注于机票服务领域的智能体，通过自然对话方式为用户提供一站式机票相关服务，无需复杂操作，即可轻松完成行程规划与票务管理。

## ✨ 核心功能

### 🔍 机票智能查询

- 支持出发地、目的地、出行日期精准查询
- 可筛选经济舱 / 商务舱 / 头等舱、直飞 / 中转
- 返回航班信息、起降时间、价格、航司等完整信息
- 支持多日期比价、低价提醒

### 🎫 机票预订

- 自然语言下单，无需填写复杂表单
- 乘客信息自动识别与校验
- 订单确认、退改规则告知
- 支持订单提交与支付引导

### 📦 订单信息查询

- 按手机号 / 身份证号查询个人订单
- 展示订单状态、航班信息、座位信息
- 支持未出行 / 历史订单分类展示
- 提供电子票号、退改签入口

### 💬 智能问答

- 航班动态、行李额度、退改政策解答
- 机场交通、值机时间等出行辅助咨询
- 多轮对话，上下文理解，流畅交互

## 🧠 技术特点

- 基于 LLM 大模型，支持自然语言交互
- 上下文记忆，支持多轮连续对话
- 接口对接真实机票数据源（可配置）
- 结构化数据解析，保证信息准确可靠
- 支持高并发、轻量化部署
